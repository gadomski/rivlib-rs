Read points from Riegl's rxp format.

You need to have RiVLib on your system in order to use this library. Get RiVLib from the Riegl website.

rxp is a data layout specification that Riegl uses for streaming discrete point data. The format is used both to store data in files and to stream data over a network.

Riegl does not publish the specification for the rxp format, so we must use their RiVLib to read rxp data.
